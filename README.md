# Resume Progress Platform
Resume Progress Platform — личная система профессионального развития для IT-специалистов. (Стремлюсь к этому, по крайней мере)

Платформа для отслеживания профессионального развития, навыков и технического прогресса.

Проект объединяет:
- интерактивное резюме,
- dashboard профессионального роста,
- систему учёта учебных часов,
- трекинг навыков,
- прогресс чтения технической литературы,
- управление проектами и учебными сессиями.

## Основные возможности

### Resume Landing
- публичная страница-резюме;
- отображение опыта, проектов и навыков;
- визуализация профессионального стека.

### Progress Dashboard
- статистика потраченных часов;
- прогресс изучения навыков;
- графики активности;
- прогресс чтения литературы;
- отображение roadmap развития.

### Skills Management
- добавление и редактирование навыков;
- уровни освоения;
- поднавыки;
- привязка навыков к проектам и учебным сессиям.

### Learning Sessions
- учёт времени обучения;
- трекинг практики;
- журнал активности.

### Future Features
- desktop timer;
- автоматическое отслеживание приложений;
- экспорт HR-резюме в PDF;
- AI-анализ пробелов в знаниях;
- интеграция с GitHub API.
